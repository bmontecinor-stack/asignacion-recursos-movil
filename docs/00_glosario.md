# Glosario / Definiciones

## Conceptos
- **FLO (Segmentos):** base operacional mensual que actúa como sábana madre.
- **GGTT:** canal con cierres mensuales que se consolidan aparte.
- **QNP:** base asociada a calidad/no pago (se integra para curvas/ajustes).
- **Curvas de permanencia:** factores para proyectar comportamiento e ingreso a 12 meses.
- **PRE_ING:** base “pre-ingresos” enriquecida antes de aplicar curvas finales.

## Campos críticos (ejemplos)
- **Mes (numérico) / id_mes:** período de consolidación.
- **Precio A.R:** precio/ingreso base utilizado en proyección.
- **Curva Ingreso 12m (Canales):** factor por canal para proyección de ingreso.
- **Curva Ingresos 12m (Segmentos):** factor por segmento para proyección de ingreso.
- **Ingreso Churn 12M:** componente de churn a 12 meses.
- **Ingreso Actividad 12M:** curva × precio.
- **Ingreso Actividad 12M TOTAL CANAL/SEGMENTO:** ingreso actividad + churn según nivel.
